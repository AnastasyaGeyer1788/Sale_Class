# 🛍️ Sale_Class
Проект реализует систему управления товарами и категориями для интернет-магазина электроники с использованием объектно-ориентированного программирования на Python.

## 🏗️ Архитектура классов

### Класс Product

**Расположение:** `src/product.py`

**Назначение:** Представляет товар в магазине

**Атрибуты:**
- `name` (str) - название товара
- `description` (str) - описание товара
- `price` (Union[int, float]) - цена товара
- `quantity` (int) - количество на складе

**Конструктор:**
```python
def __init__(self, name: str, description: str, price: Union[int, float], quantity: int)
```
### Класс Category

**Расположение:** `src/category.py`

**Назначение:** Представляет категорию товаров

**Атрибуты:**
- `name` (str) - название категории
- `description` (str) - описание категории
- `products` (list[Product]) - список товаров в категории

**Статические атрибуты:**
- `category_count` (int) - общее количество категорий
- `product_count` (int) - общее количество товаров

**Конструктор:**
```python
def __init__(self, name: str, description: str, products: list[Product])
```
## 🚀 Функциональность
### 📊 Подсчет статистики
1. [ ] Автоматический подсчет общего количества категорий
2. [ ] Автоматический подсчет общего количества товаров
